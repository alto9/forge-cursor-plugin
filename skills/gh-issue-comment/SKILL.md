# GH Issue Comment

Use the provided script to post a comment on a GitHub issue.

## Usage

Run the script: `scripts/gh-issue-comment.sh <issue-ref> <comment>`

- `issue-ref` supports issue URL, `owner/repo#123`, or issue number.
