# GH Get Milestones

Use the provided script to fetch GitHub milestones and print JSON.

## Usage

Run the script: `scripts/gh-get-milestones.sh [state] [per_page]`

- `state`: `open`, `closed`, or `all` (default: `all`)
- `per_page`: API page size (default: `100`)
